# Portfolio Website

Personal portfolio built with Next.js, TypeScript, Tailwind CSS, Sanity, and Vercel.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Sanity CMS
- Vercel deployment target
- Resend-ready contact API

## Local setup

1. Copy `.env.example` to `.env.local`
2. Install dependencies with `npm install`
3. Run the app with `npm run dev`
4. Run Sanity Studio with `npm run sanity`
